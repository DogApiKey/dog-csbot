CSBot Widget Preview

This page demonstrates the embeddable chat widget. Click the chat button in the bottom-right corner.

Embedding

Add the widget to any page with just two lines:

<script src="https://your-domain/csbot-widget.js"></script>
<csbot-widget api-url="https://your-domain/api"></csbot-widget>

Configuration

<csbot-widget
  api-url="https://your-domain/api"
  title="Support Chat"
  greeting="Hi! How can I help you?"
></csbot-widget>
